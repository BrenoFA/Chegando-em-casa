Manual

Passo a passo para utilização de um caixa eletrônico para saque.

  1. Vá presencialmente a um caixa eletrônico do banco ou credenciado;
  2. Insira o cartão de crédito do qual deseja sacar o dinheiro;
  3. Selecione a opção de saque;
  4. Informe os dados necessários;
  5. Digite o valor desejado (lembre-se que precisa ser uma quantia disponível do seu limite);
  6. Aguarde enquanto o caixa eletrônico faz a contagem das cédulas;
